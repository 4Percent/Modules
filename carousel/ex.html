			var crs = new SimpleCarousel(container,{
				btns:true,		//可空,使用内置按钮，效果如图
				easing:'easeOut',		//可空,运动模式,使用了Tween的函数,包括easeOut easeIn bounce linear
				btnMode:'horizontal',		//内置按钮对应的模式,如果使用了内置按钮则必须填写
				tabs:'horizontal',		//可空,使用内置tab,内置tab包含横排版(horizontal,效果如图)和竖排版(vertical)
				tabMode:'cover',		//内置tab对应的模式,如果使用了内置tab则必须填写

				autoPlayInterval:1500,		//自动播放间隔,如果开启了自动播放则必须填写
				duration:2000		//可空,每个动画的持续时间
			});
			crs.addTabs(document.getElementById('tab'),'horizontal','act');		//增加一个新的tab,三个参数是tab的包裹容器、对应模式、当前项的class
			crs.addBtn(document.getElementById('btn'),'vertical',-1);		//增加一个按钮,最后一个参数为-1时点击向前,为1时点击向后

CSS3版

10种模式如上,为啥有的是动词,有的是形容词,有的是名词?一目了然的名字不好起啊!

已知问题:有些模式在不同版本的Chrome下会出现各种奇怪效果。Firefox和Edge正常

			若使用内置样式和按钮,Css文件需手动引入
			示例布局:
			<div class="container m-s-crs" id="container2">
				<a href="#1" class="m-crs-card"><img src="../imgs/1.jpg" alt=""></a>
				<a href="#2" class="m-crs-card"><img src="../imgs/2.jpg" alt=""></a>
				<a href="#3" class="m-crs-card"><img src="../imgs/3.jpg" alt=""></a>
				<a href="#4" class="m-crs-card"><img src="../imgs/4.jpg" alt=""></a>
			</div>
			js:
			var crs = new Css3Carousel(container2,{
				tabs:'horizontal',
				tabMode:'flipover',
				cols:10,		//针对一些分块的效果设定的属性,如果使用则必须指定
				rows:5,		//同上
				duration:1000,		//有些模式的运动时间会长于指定值
				perspective:1000,		//可空,默认1000,景深,即CSS3的景深
				newWindow : true,		//可空,点击链接时是否在新窗口中打开链接,有些模式会使原有的链接不可操控,指定false或不指定都会在当前窗口打开
				autoPlay:'randomMode',		//随机模式,效果如上
				autoPlayInterval:2000
			});
			crs.stop();		//停止自动播放
			crs.restart();		//重启自动播放
